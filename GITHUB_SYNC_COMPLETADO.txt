================================================================================

   ─────────────────────────────────────────────────────────────────
   git clone https://github.com/sarrigui1/iBot.git
   cd iBot

2. SELECCIONAR RAMA
   ─────────────────────────────────────────────────────────────────
   Para desarrollo del instalador:
     git checkout iBot-Installer

   Para versión estable:
     git checkout main

3. INSTALAR DEPENDENCIAS
   ─────────────────────────────────────────────────────────────────
   pip install -r config/requirements.txt

4. EJECUTAR EL INSTALADOR
   ─────────────────────────────────────────────────────────────────
   python tools/advanced_installer.py "C:\iBot_Trading"

5. O EJECUTAR LA APP DIRECTAMENTE (sin instalar)
   ─────────────────────────────────────────────────────────────────
   # Primero edita config/config.ini con tus credenciales
   streamlit run src/app.py

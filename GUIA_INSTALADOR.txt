COMO USAR EL INSTALADOR

PASO 1: Preparar la instalación
────────────────────────────────────────────────────────────────────

Necesitas:
- Python 3.8 o superior instalado
- Terminal/CMD con acceso a python
- Carpeta de destino vacía (ej: C:\iBot_Trading)

PASO 2: Ejecutar el instalador
────────────────────────────────────────────────────────────────────

Abre CMD o PowerShell y ejecuta:

  cd c:\ruta\al\proyecto

Donde "C:\iBot_Trading" es la carpeta donde instalará la app.

================================================================================
QUE HACE CADA PASO
================================================================================

1. VERIFICAR PYTHON
   └─ Comprueba que tienes Python 3.8+
   └─ Si no: Instala Python desde python.org

2. CREAR DIRECTORIOS
   └─ Crea carpetas: src/, core/, config/, data/, docs/, tools/

3. COMPILAR CÓDIGO
   └─ Convierte .py a bytecode .pyc
   └─ Protege el código fuente
   └─ El usuario NO puede ver el código

4. COPIAR CONFIGURACION
   └─ config.ini plantilla
   └─ ESTO es lo único que edita el usuario

5. COPIAR DOCUMENTACION
   └─ README, guías de uso
   └─ Información de licencias

6. COPIAR DEPENDENCIAS
   └─ requirements.txt con librerías necesarias

7. INSTALAR LIBRERIAS
   └─ pip install -r requirements.txt
   └─ Descarga: streamlit, MetaTrader5, anthropic, etc.

8. CREAR LAUNCHER
   └─ launcher.bat para ejecutar fácilmente desde Windows
   └─ Solo doble clic para iniciar

9. GENERAR README
   └─ PRIMEROS_PASOS.txt con instrucciones post-instalación
   └─ El usuario abre esto primero

10. CREAR RESUMEN
    └─ installation_summary.json con metadata
    └─ Fecha, duración, versión Python

================================================================================
TROUBLESHOOTING
================================================================================

Error: "Python not found"
  └─ Instala Python desde: python.org
  └─ Asegúrate de seleccionar "Add Python to PATH"

Error: "Permission denied" en C:\iBot_Trading
  └─ Usa una carpeta donde tienes permisos (ej: Documents, Desktop)
  └─ O ejecuta CMD como administrador

Error: "pip command not found"
  └─ Reinstala Python con "Add Python to PATH"
  └─ O ejecuta: python -m pip install --upgrade pip

Error: "ModuleNotFoundError: No module named 'streamlit'"
  └─ Ejecuta manualmente: pip install -r requirements.txt
  └─ Espera a que termine

La app no abre en el navegador
  └─ Revisa el archivo: installation.log
  └─ Busca mensajes de ERROR
  └─ Ejecuta desde terminal: streamlit run app_main.py

================================================================================
CARACTERISTICAS DE SEGURIDAD
================================================================================

✓ Código compilado a bytecode
  └─ El usuario no puede ver el código fuente (.py)
  └─ Solo ve bytecode compilado (.pyc)

✓ Configuración aislada
  └─ Solo config.ini es editable
  └─ Resto del código es inmutable

✓ Sin credenciales en el código
  └─ Credenciales MT5 en config.ini (usuario)
  └─ Credenciales de desarrollo nunca distribuidas

✓ Validación de licencias en servidor
  └─ Google Apps Script endpoint privado
  └─ No se expone Google Sheet

✓ Caché offline
  └─ Funciona 7 días sin internet
  └─ Después necesita validar en servidor

================================================================================
DISTRIBUCION A CLIENTES
================================================================================

Para vender/distribuir iBot:

1. Ejecutar el instalador en tu máquina:
   python tools/advanced_installer.py "C:\dist\iBot_Trading"

2. Comprimir la carpeta:
   C:\dist\iBot_Trading → iBot_Trading.zip

3. Distribuir iBot_Trading.zip al cliente

4. Cliente ejecuta:
   - Descomprime iBot_Trading.zip
   - Ejecuta launcher.bat
   - Edita config.ini con sus credenciales
   - ¡Listo!

El cliente NO necesita Python (solo si quiere ejecutar desde terminal)

================================================================================
COMPILAR A EXE (OPCIONAL)
================================================================================

Para crear un ejecutable instalador (.exe):

  pyinstaller --onefile --windowed ^
    --add-data "src:src" ^
    --add-data "core:core" ^
    --add-data "config:config" ^
    --add-data "docs:docs" ^
    tools/advanced_installer.py

Esto genera: dist/advanced_installer.exe

Cliente solo hace doble clic en el .exe y listo.

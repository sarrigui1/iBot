================================================================================
CARACTERISTICAS DE SEGURIDAD IMPLEMENTADAS
================================================================================

1. COMPILACION A BYTECODE
   └─ Código .py → .pyc (bytecode)
   └─ Usuario NO puede ver código fuente
   └─ Imposible modificar sin recompilar

2. AISLAMIENTO DE CONFIGURACION
   └─ Solo config.ini es editable
   └─ Resto del código es inmutable
   └─ Credenciales del usuario en config.ini

3. ENDPOINT PRIVADO
   └─ Google Apps Script (no Google Sheets)
   └─ Solo devuelve JSON (no permite modificar)
   └─ Google Sheet sigue siendo privada

4. NO DISTRIBUIR SECRETOS
   └─ API keys internas NO en código
   └─ Credenciales de desarrollo NO distribuidas
   └─ Solo usuario agrega sus propias credenciales

5. VALIDACION EN SERVIDOR
   └─ Licencias validadas contra endpoint
   └─ No se puede falsificar licencia
   └─ Caché offline por seguridad
